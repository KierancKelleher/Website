# Kieran Portfolio (static)

This is a single-file, fast static portfolio you can host anywhere (GitHub Pages, Netlify, Vercel, etc.).

## Quick preview (local)
1. Download and unzip the project.
2. Open `index.html` in your browser (double-click).

## Host on GitHub Pages (simple)
1. Create a new repo (e.g., `kieran-portfolio`).
2. Upload the contents of this folder (make sure `index.html` is at the repo root).
3. In GitHub: Settings → Pages → Deploy from branch → `main` / `(root)`.
4. Your site will be live at a GitHub Pages URL.

## Customize
- Edit `index.html`:
  - Update the headline / summary
  - Add project links (GitHub, demos)
  - Add a headshot (optional): place an image in `assets/` and reference it in the hero card
